  1. DESCRIPCION

    2. Discusion de conceptos de diseno y desarrollo de paginas Web. Aplicacion de los

    lenguajes HTML y JavaScript conjuntamente con CSS para la creacion de paginas

    Web. Creacion y adaptacion de elementos de multimedios para ser utilizados en

    paginas interactivas. Requiere 45 horas de conferencia-laboratorio. 3 creditos
  1. OBJETIVOS

    2. Se espera que al finalizar el curso, el estudiante pueda:

    1. Discutir conceptos de diseno y desarrollo de paginas Web.

    2. Aplicar los lenguajes HTML y JavaScript conjuntamente con CSS para la creacion de paginas Web.

    3. Crear y adaptar elementos multimedios para ser utilizadas en paginas interactivas.

    4. Entender las implicaciones eticas, sociales y legales inherentes a la disciplina.

  1. CONTENIDO TEMATICO
    1. Conocer los editores del lenguaje HTML.
      1. Historia de los editores de HTML
      2. Introduccion a HTML.
        1. Listas numeradas, no numeradas y de definicion
        2. Enlaces internos y externos
        3. Incorporacion de imagenes, tablas y videos
        4. Diseno de formularios e incorporacion de "meta elements"
    2. Creacion de Paginas Interactivas
      1. Creacion de paginas interactivas utilizando un editor de HTML
      2. Publicacion de paginas interactivas
    3. Introduccion a CSS
      1. Uso de CSS
      2. Reglas de CSS
      3. Incorporacion de CSS a una pagina interactiva
    4. Introduccion a JavaScript
      1. Definicion
      2. Incorporacion de un JavaScript a paginas interactivas.
      3. Integracion de elementos visuales que utilizan animacion vectorial
